## 前端发展趋势

放眼前端的发展轨迹，从最早的简单静态页面，再到动态页面，再到之后复杂的 Web 应用，互联网也从最早的少量军用发展到现在的人人皆用。  
 这离不开硬件性能的发展，更高的硬件性能给浏览器提供了更多的空间去实现大规模应用。
功能完备而便捷的 Web 应用又给浏览器带来了更多的用户，海量的用户带来的高并发访问与大批量数据又在促进着分布式系统的发展，  
 而这些分布式系统又需要良好的物理设备支撑，从而又促进了硬件的发展。可见这完美的形成了闭环，而其中的核心就是人（用户）。

随着用户量的增多，应用的增多，作为管理/开发/维护者的我们需要参与的项目就会越多。  
 当你参与的项目愈来愈多后，你会发现很多功能是可以进行抽离共用的。
