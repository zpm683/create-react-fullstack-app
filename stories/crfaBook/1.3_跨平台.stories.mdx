## 跨平台

作为软件开发者我们都希望，一套代码处处运行。再不济，也希望可以一次学习，到处编写。
我们希望自己的代码能够在各种平台上运行。基于以上诉求现在跨平台技术越来越多，
就以 React 技术栈为例，ReactNative/Electron 等技术，  
为 React 代码在 App 以及桌面端运行带来了可能。其背后的原理其实就两种

- 利用 chrom 内核运行 html/css/js。
- 动态的编译为其他平台可运行的二进制文件。
