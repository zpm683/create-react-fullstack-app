## JSX

JSX 是 React 特有的概念。它是一种语法糖！
它通过仿 HTML 的语法，让你能够快速的打造组件！

#### JS 原始写法

```javascript
// 子组件1
const child1 = React.createElement("li", null, "Text1");
// 子组件2
const child2 = React.createElement("li", null, "Text2");
// 合成父组件
const root = React.createElement(
  "ul",
  { className: "my-list" },
  child1,
  child2,
);
```

#### 使用 JSX 语法糖写法

```javascript
// 父组件
const root = () => {
  <ul className="my-list">
    <li>Text1</li>
    <li>Text2</li>
  </ul>;
};
```

#### 不难看出，使用 JSX 大大提高了可读性！
