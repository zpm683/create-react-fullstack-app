### 环境变量获取

---

- isProduction() 判断是否为生产环境
- isDevelopment() 判断是否为开发环境
- isTest() 判断是否为测试环境
- getLogServerURL() 获取日志上传服务器地址
- getApiServerURL() 获取 API 服务器地址
- getApiTimeout() 获取 API 超时设定值
